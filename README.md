# Python Projects

A collection of beginner-level Python scripts. Use them to learn, explore, and practice core Python concepts.

## Repository Structure

- `Level 0/` — Beginner projects

## Level 0 — Projects

- [NumberGuessGame.py](https://github.com/Faisal-Haroon/Python-Projects/blob/main/Level%200/NumberGuessGame.py)  
  Guess a random number within a given range. Practices loops, conditionals, and input handling.

- [basicToDoList.py](https://github.com/Faisal-Haroon/Python-Projects/blob/main/Level%200/basicToDoList.py)  
  Simple command-line to-do list. Add, view, and remove items. Practices lists, functions, and basic CRUD.

- [simpleCalculator.py](https://github.com/Faisal-Haroon/Python-Projects/blob/main/Level%200/simpleCalculator.py)  
  Basic calculator for addition, subtraction, multiplication, and division. Practices arithmetic operations and error handling.

- [rockPaperScissors.py](Level%200/rockPaperScissors.py)  
  Classic Rock-Paper-Scissors game against the computer. Tracks score and validates user input.

## How to Run

Make sure you have Python 3 installed. From the repository root:

```bash
# Navigate into Level 0
cd "Level 0"

# Run any project file, e.g. Rock-Paper-Scissors
python rockPaperScissors.py

# Other examples:
python NumberGuessGame.py
python basicToDoList.py
python simpleCalculator.py
```
